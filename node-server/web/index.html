Useful bookmarks to elasticsearch FFF monitoring

  • F3Mon (Node.js version)
    Latest and greatest page with run information, stream rates, merging completion and microstate monitoring.

  • F3Mon (legacy version) with full detail microstate chart
    Warning: can overload browser if HLT menu has many modules

  • BigPic
    Monitoring of appliance resources and elasticsearch status

  • SmallPic
    Summary monitoring of the FFF system

  • MergerPlots
    Monitoring of merging and transfer latencies

Note: if you are loading any *directory* link through the cmsdaq0 proxy, make sure to add a trailing slash character to the URL, otherwise links will be broken (links from this page are correct).
