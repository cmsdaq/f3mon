* Explanation of CMSSW job exit codes
* code 127 is Unix shell missing command exit code, resulting usually from missing CMSSW installation
* code 66 can result from squid/frontier access problem. Last column estimates this type of error.
   It is however possible that not all such cases are detected and could get assigned to "code 66" column
* code 90 results from Python configuration error (sometimes from manual ramdisk cleanup script action)
* signals 9 (SIGKILL) is usually sent by manual Filter Farm cleanup procedures or OOM killer
* signal 15 (SIGTERM) is usually sent by manual cleanup procedures
